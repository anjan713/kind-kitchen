# Approved Swap Catalog — Internal Demo Policy

This is a design-time catalogue, not a medical or clinical clearance. It is deliberately **not rendered as a user-facing choice list**. `SAFE_CHOICES` and `SUBSTITUTIONS` use the same IDs.

| Ingredient | Fallback ID | Other approved IDs |
|---|---|---|
| canned crushed tomatoes | `no-salt-added-crushed-tomatoes` | `no-salt-added-diced-tomatoes`, `fresh-tomato-puree` |
| heavy cream | `fat-free-evaporated-milk` | `skim-milk-cornstarch`, `plain-nonfat-greek-yogurt` |
| vegetable broth | `no-salt-added-vegetable-broth` | `water-bay-leaf`, `homemade-unsalted-vegetable-stock` |
| olive oil | `water-saute` | `no-salt-added-broth-saute`, `dry-saute` |
| salt | `omit-salt-lemon` | `salt-free-italian-seasoning`, `garlic-onion-herbs` |

For packaged items, the UI retains a label-check note where relevant. No catalog option should be described as universally appropriate for a person’s care plan.
